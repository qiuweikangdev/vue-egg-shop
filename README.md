# vue-egg-shop

vue.js+egg.js+mysql 移动端商城

# 前端技术栈

```
Vue2.x 全家桶 、Vant 、Axios、Sass、SVG
```

# 后端技术栈

```
Node.js +Egg.js +MySQL
```

# 功能模块

1、登录注册、验证码、上传头像、个人中心、token 验证、购物车、添加商品等
